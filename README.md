## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ShuaibuPassionateProgrammer/Lumin.git
   cd Lumin
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Set up environment variables:**
   - Copy `.env.example` to `.env` and fill in required values.

4. **Run the project:**
   ```bash
   npm run start
   # or
   npx expo start
   ```
